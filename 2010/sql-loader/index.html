SQL*Loader 사용법 (초간단 정리)

간단히 참조할 용도로 정리한다. 자세한 사용법은 매뉴얼을 참조해야 한다.

1. 데이터를 로드할 테이블 생성(또는 확인)

2. 데이터 파일 확인 (xxx.dat)

3. 컨트롤 파일 작성 (xxx.ctl)

4. 데이터 로드 실행

$ sqlldr userid=xxx/xxx control=xxx.ctl

5. 로드 결과 확인

SQL> select * from xxx;
