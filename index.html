2017-12-31 on Emacs

Emacs: MMM-Mode

MMM Mode는 한 버퍼 안에서 여러 메이저 모드를 사용할 수 있게 해주는 Emacs 마이너 모드로, MMM은 Multiple Major Mode를 뜻한다.

Emacs는 버퍼마다 메이저 모드를 정할 수 있다. 예를 들어 C/C++ 파일을 편집할 때는 c-mode, Java 파일을 편집할 때는 java-mode, Clojure 파일을 편집할 때는 clojure-mode가 메이저 모드가 된다. 파일을 편집하는 데 필요한 문법 강조, 들여쓰기, 템플릿 사용, 단축키 매핑 등 여러 기능이 메이저 모드에 연결되어 있다.
