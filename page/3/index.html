Java 날짜 연산

Java 8에 추가된 날짜/시간 관련 클래스 중 LocalDate를 사용한 간단한 날짜 연산 방법을 정리한다. LocalDate에는 시간 정보가 포함되어 있지 않다. 날짜와 시간을 함께 다루려면 LocalDateTime을 사용해야 한다. UTC 오프셋이나 시간대도 함께 필요한 경우는 OffsetDateTime 또는 ZonedDateTime을 사용해야 한다.
