Emacs: MMM-Mode

MMM Mode는 한 버퍼 안에서 여러 메이저 모드를 사용할 수 있게 해주는 Emacs 마이너 모드로, MMM은 Multiple Major Mode를 뜻한다.

Emacs는 버퍼마다 메이저 모드를 정할 수 있다. 예를 들어 C/C++ 파일을 편집할 때는 c-mode, Java 파일을 편집할 때는 java-mode, Clojure 파일을 편집할 때는 clojure-mode가 메이저 모드가 된다. 파일을 편집하는 데 필요한 문법 강조, 들여쓰기, 템플릿 사용, 단축키 매핑 등 여러 기능이 메이저 모드에 연결되어 있다.

그런데 메이저 모드만으로는 부족한 경우가 있다. 예를 들어 HTML 파일에 포함된 JavaScript 코드를 편집하는 경우를 생각할 수 있다. HTML 파일이니 html-mode가 메이저 모드가 될 것이고 JavaScript 코드 부분은 문법 강조나 들여쓰기가 지원되지 않을 것이다. JavaScript 코드를 위해 메이저 모드를 js-mode(또는 js2-mode)로 설정하면 HTML 부분의 편집이 불편할 것이다.

이럴 때 mmm-mode를 사용할 수 있다. <script> 태그 안은 js2-mode를 사용하도록 설정하면 HTML과 JavaScript가 섞여 있는 파일도 편리하게 편집할 수 있다. mmm-mode를 사용하면 한 버퍼 안에서도 특정 부분을 원하는 메이저 모드로 바꿀 수 있게 된다.

블로그 글을 쓸 때 markdown-mode를 사용하는데, 소스 코드나 수식이 들어가는 경우가 자주 있다. 요즘은 주로 Clojure 코드를 많이 넣으므로 Clojure 코드가 나오는 부분은 clojure-mode로, 수식이 나오는 부분은 tex-mode로 하면 글 쓸때 편리할 것 같다.

설치 및 초기 설정 방법은 MMM Mode에 잘 나와 있다. 여기서는 markdown-mode에 clojure-mode와 tex-mode를 추가하는 방법만 정리한다.

markdown-mode에 mmm-mode 적용. 수식, Clojure 코드 하이라이팅 및 편집 기능 추가.
