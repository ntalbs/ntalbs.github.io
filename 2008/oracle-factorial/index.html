오라클에서 factorial 구하기

오라클에 factorial 함수가 없지만 다음과 같이 구할 수 있다.

select exp(sum(ln(level)))
from dual
connect by level <= N

원리

임을 이용하면 을 다음과 같이 표현할 수 있다.

n! =  1 * 2 * ... * n
   = exp(ln 1) * exp(ln 2) * ... * exp(ln n)
   = exp(ln 1 + ln 2 + ... + ln n)
